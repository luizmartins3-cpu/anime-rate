# Anime Rate 2.0 - Renovação Total

Este é um projeto de catálogo e avaliação de animes moderno, organizado e interativo.

## 🚀 Funcionalidades
- **Catálogo Amplo:** Mais de 30 animes populares.
- **Categorização:** Filtros por gênero (Ação, Romance, Shounen, etc.).
- **Sistema de Busca:** Pesquisa em tempo real.
- **Favoritos:** Salve seus animes favoritos (persiste no localStorage).
- **Avaliações:** Sistema de 5 estrelas com comentários.
- **Páginas de Detalhes:** Informações completas, sinopse, temporadas e trailers.
- **Perfil do Usuário:** Estatísticas de uso e rank de otaku.
- **Interface Moderna:** Tema escuro com animações suaves e responsividade total.

## 📁 Estrutura do Projeto
- `index.html`: Página inicial e catálogo.
- `pages/`: Páginas secundárias (detalhes, avaliação, favoritos, perfil).
- `css/`: Estilização global e componentes UI.
- `js/`: Lógica modularizada para cada página.
- `images/`: Recursos de imagem locais.

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3 (Variáveis, Flexbox, Grid, Animações)
- JavaScript Vanilla (ES6+)
- localStorage API
- FontAwesome para ícones

## 👤 Desenvolvedor
Desenvolvido como uma renovação completa do projeto original Anime Rate.
