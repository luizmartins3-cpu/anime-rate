# anime-rate# Meu Projeto LabSIS

## 📚 Descrição do Projeto
O Anime Rate é um site desenvolvido como atividade acadêmica com foco na utilização de HTML5 semântico.

O objetivo do projeto é apresentar análises (reviews) de animes populares, organizadas de forma estruturada e acessível.

---

## 🏗 Estrutura Utilizada (HTML5 Semântico)

O projeto foi desenvolvido utilizando as seguintes tags semânticas:

- `<header>` → Cabeçalho do site
- `<nav>` → Menu de navegação
- `<main>` → Conteúdo principal
- `<section>` → Agrupamento de conteúdo relacionado
- `<article>` → Cada review individual de anime
- `<figure>` e `<figcaption>` → Imagens das reviews
- `<time>` → Data de publicação
- `<footer>` → Rodapé do site

---

## ⭐ Funcionalidades Implementadas

- Menu de navegação interno
- Reviews organizadas em `<article>`
- Exibição de imagem para cada anime
- Data de publicação com `<time>`
- Formulário de envio de avaliação
- Validação de formulário com:
  - `required`
  - `type="email"`
  - `type="number"`
  - Limite mínimo e máximo de nota
